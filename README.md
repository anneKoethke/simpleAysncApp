# simpleAysncApp
to try out AsyncStorage in React Native + Expo CLI

## usage
In this 'very useful' app, you can save names. They'll have ids. That's about it - so nice! 

### using AsyncStorage.
- getAllKeys(), 
- multiGet(keyList), 
- setItem(key, JSONValue), 
- removeItem(key) and 
- clear()
